React Movie Database

react js project

For this project, I used React JS and The Movie DB API calls to create a dynamic movie database.

As you can see, the home page consists of a Hero image displaying the current most popular movie according to the database. It then proceeds to list all the current Popular Movies with their poster art for a kind of "Netflix" feel.

As a user you can browse the selection available, and even click on an of the films to view more information about them.

When you click on a movie, you will see some information about the movie as well as the option to watch the trailer.

If you scroll down, you can even see a list of actors and their roles in the film, accompanied by photos where available. Lastly, users also have the option to search the database for films of their choice.

As you can see, users are able to view all the Spider-Man films when searching for "Spider"
